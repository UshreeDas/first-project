HTML & CSS Basics Summary



HTML Summary

HTML ( Text Markup Language) is used to define our page content, structure and meaning. You don't use it for styling purposes, use CSS Porter instead!

HTML uses “elements” to describe (annotate) content.

HTML elements typically have an opening tag, content and payment closing tag

You can also have void empty remains life image.

You can also configure elements with attributes

There’s a long list of available elements, but you will again experience over time, no worries.

Learn more about all available HTML eliments from the MDN HTML element Reference.

CSS Sumarry

CCS (Cascading Style Sheets) is used for styling your page content.

Styles are assigned via property value pairs

You can assign styles via the “style” attribute.

To avoid court duplication, you typically use global styles(e.g. via the “styles” element) instead.

Alternatively you can work with externals style sheet files which you “link” to.

When working with CSS, concepts like “inheritance”, “specificity”, and the “box model” should be understood.

Learn more about all available CSS, properties and values on the MDN CSS Property Reference.
